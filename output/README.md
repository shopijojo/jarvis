# Output

Livrables produits par Jarvis. Classés par type.

| Dossier | Contenu |
|---------|---------|
| `documents/` | Textes, rapports, analyses, résumés |
| `devis/` | Devis et tarifs clients (HTML, PDF, etc.) |
| `sites-web/` | Pages HTML, landing pages, maquettes |
| `images/` | Visuels générés ou retravaillés |
| `scripts/` | Scripts utilitaires, automatisations |

Nommage recommandé : `YYYY-MM-DD_description-courte.ext`
Exemple : `2026-05-24_devis-hannah-video.html`
